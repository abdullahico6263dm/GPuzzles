
[1] Download GPuzzles.zip and deploy into xampp/htdocs/GPuzzles

[2] Launch XAMPP; Start Apache Server and SQL Server; 

[3] Go to SQL Admin panel (phpmyadmin); Create a database called "gpuzzles_db".

[4] Import the SQL file (xampp/htdocs/quiz_master/sql/gpuzzles_db.sql) into the database.

[5] Launch the quiz_master application
http://localhost/gpuzzles
